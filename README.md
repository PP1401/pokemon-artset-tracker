# Pokémon Artset Tracker — PWA

## Come pubblicare su GitHub Pages (gratis, 5 minuti)

### 1. Crea un account GitHub
Vai su https://github.com e registrati se non ce l'hai.

### 2. Crea un nuovo repository
- Clicca sul "+" in alto a destra → "New repository"
- Nome: `pokemon-artset-tracker` (o qualsiasi nome)
- Lascia pubblico (necessario per GitHub Pages gratis)
- Clicca "Create repository"

### 3. Carica i file
Nel repository appena creato clicca "uploading an existing file" e trascina TUTTA la cartella `pwa-pokemon`:
- index.html
- manifest.json
- sw.js
- cartella icons/ con tutte le icone

Clicca "Commit changes".

### 4. Attiva GitHub Pages
- Vai in Settings → Pages (menu a sinistra)
- Source: "Deploy from a branch"
- Branch: main → / (root)
- Clicca Save

Dopo ~2 minuti il tuo sito sarà online su:
`https://TUO-USERNAME.github.io/pokemon-artset-tracker/`

## Installa come app sul telefono

### iPhone (Safari)
1. Apri il link in Safari
2. Tocca il tasto Condividi (□↑) in basso
3. Scorri e tocca "Aggiungi alla schermata Home"
4. Dai un nome e tocca "Aggiungi"

### Android (Chrome)
1. Apri il link in Chrome
2. Tocca i tre puntini in alto a destra
3. Tocca "Aggiungi alla schermata Home" o "Installa app"
4. Conferma

L'app apparirà come icona sul telefono e si aprirà senza barra del browser!

---

## Funziona offline?
Sì! Una volta caricata la prima volta, funziona anche senza connessione.
I dati sono salvati nel telefono (localStorage) e non vengono persi.
